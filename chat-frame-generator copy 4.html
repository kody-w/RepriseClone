How to Use

  1. Upload your base HTML frame (the static page with chat messages)
  2. Use the visual preview to select example chat messages (agent and customer) to copy their styling
  3. Add your own chat messages in sequence (agent or customer)
  4. Export messages to JSON to save and edit externally, then import back to continue
  5. Generate frames - your messages will match the original styling
  6. Load frames into the animator to step through them
