How it works

  1. Get the Bookmarklet: Drag the recorder bookmark to your bookmarks bar (one-time setup)
  2. Record Any Page: Navigate to any website and click the bookmarklet to start recording
  3. Capture Interactions: All clicks, inputs, scrolling, and DOM changes are recorded
  4. Export: Save your recording as a JSON file when you're done
  5. Replay: Load the JSON file in the player to watch the demo playback with full controls
